# InteriorClarify Dataset

> A multimodal benchmark dataset for hierarchical clarification question generation in interior design consultations.

## üì¶ Dataset Overview

**InteriorClarify** contains **1,016 interior design consultation cases**. Each instance includes:

| Field | Description |
|-------|-------------|
| `client_request` | Ambiguous text request from client (avg. 28 Chinese chars) |
| `image` | Photo of existing space OR generated image from text description |
| `clarification_questions` | Designer-written questions organized by priority level |
| `missing_elements` | Annotated missing design elements with importance labels (L‚ÇÉ/L‚ÇÇ/L‚ÇÅ) |
| `room_type` | Category: living room / bedroom / kitchen / bathroom |
| `metadata` | Source platform, annotation round, language info |

## üéØ Three-Tier Importance Framework

Clarification questions are labeled with priority levels to reflect professional consultation strategy:

| Level | Label | When to Ask | Impact if Missing |
|-------|-------|-------------|------------------|
| **L‚ÇÉ** | Critical | First round | Cannot start substantive design planning |
| **L‚ÇÇ** | Influential | Second round | Core direction uncertain (layout/style) |
| **L‚ÇÅ** | Refinement | Third round | Basic plan feasible, but less personalized |

**Annotation Protocol**: For each case, designers sequentially identify missing elements at L‚ÇÉ ‚Üí L‚ÇÇ ‚Üí L‚ÇÅ levels and generate up to 3 targeted clarification questions.
